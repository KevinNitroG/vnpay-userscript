# VNPay Userscript

Auto fill VNPay card for sandbox testing

## Installation

1. [Tampermonkey](https://www.tampermonkey.net/)
2. Install [script](https://github.com/KevinNitroG/vnpay-userscript/releases/latest/download/vnpay-userscript.user.js)

## Usage

1. Open VNPay sandbox payment page
   > Test with <http://sandbox.vnpayment.vn/tryitnow/Home/CreateOrder>
2. It will auto fill

## Configuration

Currently only support this card

```
Ngân hàng: NCB
Số thẻ: 9704198526191432198
Tên chủ thẻ:NGUYEN VAN A
Ngày phát hành:07/15
Mật khẩu OTP:123456
```
